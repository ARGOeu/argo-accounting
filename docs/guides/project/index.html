Manage projects

Before you start​

You can manage a project.Firstly, you need to be registered to the accounting-system and then you need to contact the system administrator, to assign you one or more roles on the project.

ΝΟΤΕ In accounting sytem, already exists the project_admin role , that permits the user to perform any operation, on a specific project. In case the user is assigned with any other role, he can operate according to the role's permissions.

OPERATIONS​

READ the project​

Details
You can display the project's details. Apply a request to the api. For more details , see here

ASSOCIATE providers with a specific project installations to the provider​

Details
You can associate providers with a specific project. This is a required action in order, to be able to assign installations and metrics , as next actions. One or more projects can be associated with the specific project. Apply a request to the api. For more details , see here

DISSOCIATE provider from a specific project​

Details
If a provider is associated with a specific project, you can dessociate it , if no installations are assigned. Apply a request to the api For more details , see here

GET the project's hierarchical structure​

Details
You can view the hierarchy of the project (all providers and installations that are assigned to the project). Apply a request to the api. For more details , see here

GET all the projects assigned​

Details
You can view all the projects that are assigned to you. Apply a request to the api . For more details , see here

Provide access roles on the project​

Details
You can provide users with access roles on the project. Firstly you need to know the user's id. To do so, you can read the registered clients ( see here and then decide one or more roles , that this user will be assigned with on the project. Then you apply a request to the api, see here.

SEARCH projects​

Details
If you are assigned with many projects, you can search for specific project/projects, that match one or more criteria. Apply a request to the api. You need to provide the search criteria in a specific syntax. For more details,how to syntax the request, see here
In the case the role assigned to you , on the project , is administrative, you can perform all the actions on the providers , described at this section, as well as all the actions on installations , described at this section
